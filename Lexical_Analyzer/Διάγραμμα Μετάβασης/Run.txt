Το Διάγραμμα Μετάβασης είναι το TD.jff και το κάναμε με το εργαλείο JFLAP7.1.jar

Για να ανοίξουμε το JFLAP7.1.jar και το TD.jff κάνουμε τα εξής:

Για Ubuntu:
	Τρέχουμε στο φάκελο τον αρχείων στο τερματικό:
	1)	sudo apt install default-jre
	2)	java -jar JFLAP7.1.jar
	Στη συνέχεια:
	3)	Επιλέγουμε Finite Automaton
	4)	Επιλέγουμε File --> Open...
	5)	Εντοπίζουμε το αρχείο TD.jff και το ανοίγουμε στο JFLAP

Για Windows:
	Στο φάκελο τον αρχείων
	1)	Διπλό κλίκ στο JFLAP7.1.jar
	2)	Επιλέγουμε Finite Automaton
	3)	Επιλέγουμε File --> Open...
	4)	Εντοπίζουμε το αρχείο TD.jff και το ανοίγουμε στο JFLAP

(Σημείωση 1) : Το αυτόματο είναι σε μορφή ΜΠΑ και όχι ΝΠΑ όπως μας ζητείτε.
(Σημείωση 2) : Η εργασία του μεταγλωττιστή έχει υλοποιηθεί σε Ubuntu 18.04 και η λειτουργηκότητά του έχει ελεγχθεί εκεί.
